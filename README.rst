medialog.bikebooking
==============================================================================

Add on to book bikes

Features
--------

- Sends email for user to confirm


  
Installation
------------

Install medialog.bikebooking by adding it to your buildout::

    [buildout]

    ...

    eggs =
        medialog.bikebooking


and then running ``bin/buildout``
